# Dig World

3D воксельная мульти‑игра в браузере. Копай землю, ищи руды, отбивайся от монстров, покупай улучшения. Играй с другом через **точку доступа Wi‑Fi** — никаких установок приложений, всё работает в браузере.

## Что есть в игре

- 🌍 Воксельный мир 64×64×64 с холмами, водой, деревьями, пещерами и шахтами.
- 🪨 Руды на разной глубине: уголь, железо, золото, алмаз.
- ⛏️ **Лопата** копает землю/песок, **кирка** — камень и руды. Обе выдаются сразу.
- 🛒 Магазин у золотого блока (на спавне): продаёшь руды → покупаешь улучшения.
   - 5 уровней инструментов: Деревянная → Каменная → Железная → Золотая → Алмазная.
- 👾 Монстры спавнятся в пещерах, преследуют игроков и кусают. За убийство — монеты.
- ⚔️ PvP включён: можешь бить других игроков, за убийство получаешь половину их монет.
- 💾 Мир и прогресс игроков сохраняются на диск автоматически (по имени игрока).

## Запуск

Нужен **Node.js 18+**.

```bash
npm install
npm start
```

Сервер напечатает IP‑адреса, например:

```
=== DIG-WORLD ===
Сервер запущен на порту 3000.
Открой одну из ссылок ниже в браузере телефона:
  http://localhost:3000/   (с этого устройства)
  http://192.168.43.1:3000/   (через wlan0)
```

## Как играть с другом через точку доступа

1. На одном телефоне или ноуте запусти сервер (`npm start`).
2. Раздай Wi‑Fi с этого устройства (или подключите все устройства к одной общей сети).
3. На втором телефоне открой в браузере IP из логов сервера, например `http://192.168.43.1:3000/`.
4. Введи имя → жми «Играть».

> Совет: на Android можно запустить сервер через **Termux** (`pkg install nodejs`, потом `npm start`).
> На iOS сервер запустить нельзя — используй ноутбук как хост.

## Управление

### На телефоне
- **Левый стик** — ходьба.
- **Свайп по экрану справа** — обзор камеры.
- **Кнопка «Копать»** — копает блок перед собой (удерживай для непрерывного копания); если в радиусе ‑ моб или игрок, бьёт его.
- **Кнопка «Прыжок»** — прыжок.
- **Кнопка «Положить»** — ставит выбранный в инвентаре блок.
- Кнопки сверху: **Инв.**, **Магазин**, **Лопата/Кирка** (переключение оружия), **Чат**.

### На компьютере (для отладки)
- **WASD** — ходьба, **Space** — прыжок.
- **Мышь** — обзор (клик по канвасу — pointer lock).
- **ЛКМ / F** — копать, **ПКМ / E** — поставить блок, **Q** — сменить инструмент.

## Технологии

- Сервер: Node.js, Express, Socket.IO.
- Клиент: Three.js (вендорится локально в `public/lib/three.min.js`, чтобы работать без интернета).
- Мир: один `Uint8Array` блоков, простая реалтайм‑синхронизация через WebSocket.

```

  game.js           — игровое состояние, обработчики событий

  persistence.js    — сохранение/загрузка
  constants.js      — общие константы

  lib/three.min.js  — Three.js (вендорено)
  js/

    chunkMesher.js  — построение мешей чанков
    world.js        — клиентское хранилище блоков
    player.js       — физика локального игрока
    controls.js     — тач/клава контроллер
    network.js      — Socket.IO обёртка

```

## Сетевой протокол (Socket.IO)

Клиент → сервер:
- `hello { name }` — представиться.
- `input { x, y, z, yaw, pitch }` — позиция (~10 Гц).
- `dig { x, y, z }` — попытка сломать блок.
- `place { x, y, z, resource }` — поставить блок из инвентаря.
- `attackMob { id, weapon }`, `attackPlayer { id, weapon }` — атака.
- `sell { resource, count }`, `upgrade { kind }` — магазин.
- `respawn` — после смерти.
- `chat { text }` — сообщение в чат.

Сервер → клиент:
- `init { you, world, players, mobs, state }` — стартовое состояние.
- `tick { players, mobs }` — позиции (~10 Гц).
- `block { x, y, z, type }`, `mobSpawn`, `mobDespawn`, `mobHit`.
- `inv { coins, inv, tools, hp, alive }` — приватные обновления.
- `playerJoin`, `playerLeave`, `playerHit`, `playerDied`, `playerRespawn`.
- `flash { msg }` — короткое уведомление в HUD.
- `chat { from, text }`.
